Utils
=====

hex/bin utils written in C/C++

## Build

Build using CMake: https://cmake.org/download/

For example on Windows:

```
$ mkdir build
$ cd build
$ cmake -G "Visual Studio 14 2015 Win64" ..
$ cmake --build . --config Release
```
